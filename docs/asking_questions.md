# How to Ask Questions / 如何提问

## EN

Before asking for help, provide:

1. Goal and expected behavior.
2. Exact command/script and parameters.
3. Input instance summary (size, constraints).
4. Solver/model output and objective value.
5. Feasibility status and violations.
6. Error log or traceback (if any).
7. What you already tried.

Template:

```text
Goal:
Environment:
Instance:
Command:
Expected:
Actual:
Error/Log:
Tried:
Attachments:
```

## 中文

提问前请提供：

1. 目标与预期行为。
2. 完整命令/脚本与参数。
3. 输入算例摘要（规模、约束）。
4. 求解器/模型输出与目标值。
5. 可行性状态与违约情况。
6. 报错日志或 traceback。
7. 已尝试的排查方法。

模板：

```text
目标：
环境：
算例：
命令：
预期结果：
实际结果：
报错/日志：
已尝试：
附件：
```
